Aprendiendo Git

Indicar que vamos a iniciar a trabajar con git en nuestro proyecto

git init (una sola vez por proyecto)

Para saber a que archivos no se le esta dando seguimiento

git status -s

Para hacer un seguimiento

git add (nombre del archivo)

git add index.html

Cuando veas una A en el proyecto significa que se le esta dando seguimiento

Para ahcer el seguimiento de todos:

solo tienes que poner git add .

Antes de seguir posiblemnte te pida una identificacion

solo tienes que poner git config --global user.email "correo"

Para llevar al repositorio local

git commit -m (y un mensaje com descripcion)
